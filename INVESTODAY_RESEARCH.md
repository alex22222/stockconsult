## 三、已验证的 API 数据结构（从项目代码提取）

### 1. `get_stock_score` — 综合评分
```json
{
  "score": 72.5,           // 综合得分 0-100
  "skillScore": 68.3,      // 技术面得分
  "emotionScore": 75.1,    // 情绪面得分（新闻+社交媒体）
  "financeScore": 71.2,    // 基本面得分
  "industryScore": 76.5,   // 行业景气度得分
  "scoreAvg": 72.5         // 平均分
}
```
**用途**：一键量化选股、多维度风险评估、行业对比

### 2. `list_entity_related_news` — 新闻情绪
```json
[
  {
    "title": "...",
    "content": "...",
    "publishTime": "2026-05-21",
    "sentimentScore": 0.75,   // 情绪得分 -1~+1
    "source": "财联社"
  }
]
```
**用途**：NLP 情绪时间序列、事件驱动策略、舆情监控

### 3. `get_stock_finance_valuation` — 估值排名
```json
{
  "peTtmRk": 35.2,         // PE-TTM 行业排名（越小越好）
  "peTtmRkHist": 42.1,     // PE 历史分位
  "pbRk": 28.5,            // PB 行业排名
  "psRk": 45.3,            // PS 行业排名
  ...
}
```
**用途**：相对估值判断、均值回归策略、价值选股

### 4. `get_stock_finance_profit_ability` — 盈利排名
```json
{
  "roeRk": 15.2,           // ROE 行业排名
  "grossMarginRk": 22.1,   // 毛利率排名
  "netMarginRk": 18.5,    // 净利率排名
  ...
}
```
**用途**：盈利质量筛选、护城河评估

### 5. `list_stock_performance_metrics` — 表现指标
```json
{
  "consecutiveChangeDays": 3,     // 连续涨跌天数
  "consecutiveLimitDays": 0,      // 连续涨停天数
  "isHistoricalHighLow": 1,       // 是否历史新高
  "isYtdHighLow": 1,              // 是否年内新高
  "isMtdHighLow": 0,              // 是否月内新高
  "isWtdHighLow": -1              // 是否周内新低
}
```
**用途**：反转信号（涨太多要跌？）、动量延续判断、异常波动检测

## 四、可以形成的有用报告

基于 investoday 的 200+ API，可以构建以下报告体系：

### 报告 1：每日早盘舆情简报（5 分钟生成）
**数据来源**：`list_entity_related_news` + `get_stock_score`
**内容**：
- 持仓股票新闻情绪变化（今日 vs 昨日）
- 负面新闻预警（sentimentScore < -0.5）
- 综合评分突变预警（score 变化 > 10 分）
- AI 生成操作建议

### 报告 2：量化选股雷达（每日收盘后）
**数据来源**：`get_stock_finance_valuation` + `get_stock_finance_profit_ability` + `get_stock_score`
**内容**：
- 低估值高盈利双因子筛选（peRk < 30 & roeRk < 20）
- 情绪超卖反弹候选（emotionScore < 30 & 连续下跌 > 3 天）
- 行业龙头评分排行
- 技术面+基本面综合评分 Top 20

### 报告 3：机构动向追踪（每周）
**数据来源**：`list_stock_fund_holdings` + `list_stock_research_reports`
**内容**：
- 基金增减仓变化（季度对比）
- 研报评级上调/下调列表
- 目标价 vs 当前价价差最大机会
- 北向资金/主力资金流向

### 报告 4：事件驱动机会（实时监控）
**数据来源**：`list_stock_announcements` + `get_stock_earnings_flash` + `list_entity_related_news`
**内容**：
- 业绩预告超预期（实际 > 预期 20%+）
- 重大事项公告（重组、并购、定增）
- 政策利好/利空板块影响分析
- 新闻情绪突变股（日内 sentiment 变化 > 0.5）

### 报告 5：组合健康度体检（每周）
**数据来源**：全部 API 综合
**内容**：
- 持仓股综合评分分布（ vs 全市场百分位）
- 估值热力图（PE/PB 行业排名）
- 新闻情绪温度计（整体偏多/偏空）
- 异常波动预警（连涨/连跌、历史新高/低）
- 分散度分析（行业集中度、风格暴露）

### 报告 6：量化因子有效性报告（每月）
**数据来源**：历史数据 + 量化因子 API
**内容**：
- 价值因子（低 PE/PB）近 1 月/3 月/1 年 Rank IC
- 动量因子近 期表现
- 情绪因子（新闻 sentiment）预测能力
- 复合因子组合回测收益
- 因子拥挤度预警

### 报告 7：宏观-微观联动分析（每月）
**数据来源**：宏观经济 API + 个股数据
**内容**：
- PMI/CPI/利率 与 持仓股行业的相关性
- 货币政策宽松/紧缩周期中的受益/受损板块
- 汇率波动对出口型企业的冲击测算
